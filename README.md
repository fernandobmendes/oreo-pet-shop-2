# Oreo Pet Shop 2

Projeto desenvolvido para a Fase 2 da disciplina Fundamentos de Sistemas Web.

## Funcionalidades
- Página inicial com carrossel Bootstrap
- Página de produtos com imagens, descrição e valores
- Página de serviços com banho e tosa
- Formulário de cadastro do cliente e do pet
- Agendamento com serviço, método, data e horário
- Uso de CSS, Bootstrap e JavaScript
- Recurso de acessibilidade com textos alternativos nas imagens
- Organização em arquivos HTML, CSS e JavaScript

## Tecnologias utilizadas
- HTML
- CSS
- Bootstrap
- JavaScript
