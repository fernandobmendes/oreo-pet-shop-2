# Oreo Pet Shop 2

Projeto desenvolvido para a disciplina **Fundamentos de Sistemas Web - Fase 2**.

## Descrição

O Oreo Pet Shop 2 é um sistema web para um petshop, desenvolvido com HTML, CSS, Bootstrap e JavaScript.

O site apresenta produtos, serviços, formulário de cadastro do cliente e do pet, além de agendamento para banho ou tosa com opção de tele-busca ou entrega do pet no local.

## Funcionalidades

- Página inicial com carrossel de banners
- Página de produtos
- Página de serviços
- Formulário de cadastro do cliente
- Formulário de cadastro do pet
- Escolha do serviço: banho ou tosa
- Agendamento com data e horário
- Escolha do método: tele-busca ou entrega no local
- Mensagem de confirmação usando JavaScript
- Página de contato
- Layout responsivo com Bootstrap
- Acessibilidade com uso de textos alternativos nas imagens

## Tecnologias utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- GitHub Pages

## Acessibilidade

O projeto utiliza o atributo `alt` nas imagens para auxiliar leitores de tela e melhorar a acessibilidade para pessoas com deficiência visual.
